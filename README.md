## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Production build: https://morningandevening.netlify.app/

Experimenting with:

- Next.js
- Netlify
- SVG / CSS Animations
- SCSS / CSS Modules
